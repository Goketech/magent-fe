## 🚀 Quick Start

### Prerequisites
- Node.js 22+
- npm
- Supabase Account

### Installation

1. Clone the repository
```bash
git clone https://github.com/Goketech/magent-fe.git
cd magent-fe
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
```bash
cp .env.example .env.local
```

4. Run the development server
```bash
npm run dev
```
